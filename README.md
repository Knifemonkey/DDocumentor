This is your new Play 2.1 application
=====================================

This file will be packaged with your application, when using `play dist`.


Useful commands
----------
Compile code:
```
play compile
```

Run all tests:
```
play test
```

In case something doesn't work:
```
play clean-all
```


Open project on http://localhost:9000
```
play run
```
